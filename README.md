# Budget_Tracker
🔹 📊 Budget Tracker Web App

A modern and user-friendly Budget Tracker web application built using HTML, CSS, and JavaScript. This app helps users manage their finances efficiently by tracking income and expenses, providing a clear overview of their financial status.

✨ Features
Add and delete income & expense transactions
Real-time balance calculation
Clear summary of total income and expenses
Simple and intuitive user interface
Data persistence using local storage
Responsive design for mobile, tablet, and desktop
🎯 Purpose

This project is designed to help users develop better financial habits by keeping track of their daily spending and income in an organized way.

🛠️ Tech Stack
HTML5
CSS3 (Responsive Design)
JavaScript (Vanilla JS)
💡 Future Improvements
Expense categories and filtering
Search functionality
Data visualization (charts/graphs)
Dark mode support
Export data option
⚠️ Note

This is a frontend-based project and stores data locally in the browser.
